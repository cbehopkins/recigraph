# ADR-002: YAML is the Source Language

**Status:** Accepted

## Context

Recipes need to be authored and maintained by humans.

The source format should be readable in Git, easy to review, straightforward to edit manually and capable of representing structured relationships.

## Decision

The ReciGraph DSL will be authored in YAML.

The YAML files are the single source of truth for the knowledge base.

Generated outputs (HTML, PDFs, indexes, etc.) must never be edited directly.

The YAML authoring format should prioritise readability over faithfully mirroring the internal object model. The loader is responsible for translating authored YAML into the richer in-memory representation used by the compiler.

## Consequences

Authors work with concise, human-friendly files.

The compiler retains the freedom to evolve its internal representation without changing the authoring experience.

YAML remains the only hand-authored artefact.
